## 第五步：执行交易

### 示例：招商银行买入操作

```
AI建议: 买入25%可用资金
当前价格: ¥35.20
可用现金: ¥40,000

计算:
1. 买入金额 = ¥40,000 × 25% = ¥10,000
2. 买入股数 = ¥10,000 ÷ ¥35.20 ≈ 284股
3. 取整到100倍数 = 200股 (保守) 或 300股
4. 实际花费 = 300股 × ¥35.20 = ¥10,560

执行:
- 登录券商APP
- 选择招商银行(600036)
- 下单买入 300股
- 价格：市价或限价¥35.20
```

### 风险控制

```python
设置止损位:
买入价: ¥35.20
止损价: ¥34.14 (-3%)
止盈价: ¥38.72 (+10%)
```

## ⚠️ 重要风险提示

### 1. AI模型限制

```
✅ 能做到:
- 分析技术指标
- 评估短期风险
- 提供交易参考

❌ 不能做到:
- 预测突发事件
- 理解基本面变化
- 保证盈利
```

### 2. 资金管理

```
建议配置 (10万本金):
- 单只股票: ≤ ¥20,000 (20%)
- 同时持仓: 3-4只
- 现金储备: ≥ ¥20,000 (20%)
- 总风险暴露: ≤ ¥80,000 (80%)
```

### 3. 止损规则

```
单只股票止损:
- 日内亏损 > 3%: 减仓50%
- 持仓亏损 > 5%: 清仓止损

组合止损:
- 日内亏损 > 2%: 暂停新开仓
- 周亏损 > 5%: 全面减仓
- 月亏损 > 8%: 暂停交易
```
