# 快速开始 - V8 版本

## 🚀 5 分钟快速启动

### 步骤 1: 准备数据（如果还没有）

```bash
# 如果已经有 V7 的数据，跳过此步骤
python get_etf_data_akshare.py
```

### 步骤 2: 测试环境

```bash
python test_v8.py
```

预期输出：

### 步骤 3: 生成市场情报缓存（推荐，10分钟）

```bash
python generate_intelligence.py
```

这会生成 2020-2025 年的模拟市场情报，避免训练时重复生成。

### 步骤 4: 开始训练（2-3小时）

```bash
python train_v8.py
```

**训练参数**：
- 总步数：300 万步
- 并行环境：8 个
- 预计时间：2-3 小时（取决于硬件）

训练过程中可以打开 TensorBoard 监控：

```bash
# 新开一个终端
tensorboard --logdir=./logs_v8/
# 浏览器访问 http://localhost:6006
```

## ⚙️ 配置选项

### 使用真实 LLM API（可选）

如果您想使用真实的 DeepSeek API：

**1. 获取 API Key**
- 访问 https://platform.deepseek.com
- 注册并充值（建议 10 元）
- 创建 API Key

**2. 设置环境变量**

```bash
# Windows PowerShell
$env:DEEPSEEK_API_KEY = "your_api_key_here"

# Linux/macOS
export DEEPSEEK_API_KEY="your_api_key_here"
```

**3. 重新训练**

```bash
python train_v8.py
```

现在训练会使用真实的 LLM 分析（每次训练约 2 元成本）。

```bash
$env:DEEPSEEK_API_KEY = "your_key"

from llm_market_intelligence import MarketIntelligenceAgent

agent = MarketIntelligenceAgent(provider='deepseek')

intel = agent.get_market_intelligence(today, force_refresh=True)
print(intel)
"

# 成本：< 1 元/年
# 性能：最优
```
