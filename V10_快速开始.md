# V10 版本快速开始指南

## 🚀 快速启动

### 1. 安装依赖

```bash
# 基础依赖
pip install torch pandas numpy matplotlib flask

# 可选：BERT支持（用于高级文本处理）
pip install transformers

# 可选：数据源
pip install akshare baostock tushare
```

### 2. 运行系统

```bash
python real_time_predict_v10.py
```

### 3. 访问可视化

在浏览器中打开：http://127.0.0.1:8080

## 🔧 配置调整

编辑 `real_time_predict_v10.py` 中的配置：

```python
# 启用/禁用功能
ENABLE_TRANSFORMER = True
ENABLE_MULTIMODAL = True
ENABLE_VISUALIZATION = True
ENABLE_HOLOGRAPHIC = True

# Transformer配置
TRANSFORMER_D_MODEL = 64
TRANSFORMER_NHEAD = 4
TRANSFORMER_NUM_LAYERS = 3

# 可视化配置
VISUALIZATION_PORT = 8080
```

## 📊 输出文件

- `visualization_output/dashboard.png` - 综合仪表板
- `visualization_output/dashboard.html` - HTML仪表板
- `visualization_output/data.csv` - 数据导出
- `visualization_output/price_chart_*.png` - 价格图表

## 💡 使用技巧

1. **首次运行**：系统会自动训练Transformer模型（需要一些时间）
2. **可视化访问**：确保端口8080未被占用
3. **文本数据**：可以集成新闻API或社交媒体数据源
4. **性能优化**：如果运行慢，可以减少Transformer层数或序列长度

## ⚠️ 注意事项

1. 首次运行需要下载BERT模型（如果启用），可能需要较长时间
2. 可视化功能需要matplotlib，某些环境可能需要安装图形库
3. GPU加速：如果有GPU，可以设置`use_gpu=True`加速训练
